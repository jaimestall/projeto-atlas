Listas de comandos SQL

CRUD (Create, Read, Update, Delete)

  • INSERT INTO table_name (columns_names) VALUES values;

    Insere valores em determinado campo de uma tabela

  • SELECT (col1, col2, col3) FROM table_name WHERE column_name = filter;

    Um dos comandos mais importantes em SQL, seleciona colunas de uma determinada tabela

  • UPDATE table_name SET column_name = new_value WHERE column_name = filter;

    Depois do parâmetro SET é possível haver mais de uma coluna.

  • DELETE FROM table_name WHERE column_name = filter;
    • É importantíssimo ressaltar a importância da cláusula WHERE nos comando de UPDATE e DELETE.

Database

  • CREATE DATABASE database_name;

    Cria um banco de dados

  • SHOW DATABASES;

    Exibe na tela os bancos de dados no servidor

  • USE database;

    Seleciona o banco de dados no qual se deseja trabalhar

  • DROP DATABASE database_name;

    Exclui um banco de dados

Criação de tabela e formatação de colunas

  • CREATE TABLE table_name (col1 datatype1, col2 datatype2);

    Cria uma tabela com as colunas col1 e col2, com os data types 1 e 2 respectivamente

  • DROP TABLE table_name;

    Deleta uma tabela

  • ALTER TABLE table_name ADD COLUMN column_name data_type;

    Adiciona a coluna column_name do tipo data_type na tabela table_name

  • ALTER TABLE table_name DROP COLUMN column_name;

    Remove a coluna column_name da tabela table_name

  • ALTER TABLE table_name MODIFY COLUMN column_name data_type;

    Modifica a coluna column_name da tabela table_name para virar do tipo data_type

Constraints

  • NOT NULL

    Impede que determinado campo seja nulo

  • INDEX

    CREATE INDEX index_name ON table_name(column_name)

  • UNIQUE

    Impede que determinado campo tenha valores iguais para registros diferentes

  • PRIMARY KEY

    (column_name)

  • AUTO_INCREMENT

    Atribui a uma coluna de uma tabela a função de se auto incrementar conforme são inseridos registros

  • FOREIGN KEY

    FOREIGN KEY (column_name) REFERENCE table_name(column_name)

Tipos de dados

  • CHAR(n)

    Campo de texto contendo até n caracteres (255 caracteres)

  • VARCHAR(n)

    Campo de texto contendo até n caracteres (65535 caracteres)

  • BOOL

    Campo booleano, aceitando 0 para falso e qualquer outro valor numerico para verdadeiro

  • INT

    Campo numérico contendo números de -2147483648 até 2147483648

  • FLOAT

    Campo numérico contendo números com vírgula

  • DATE

    Campo de data, sempre no formato 'YYYY-MM-DD'

  • DATETIME

    Campo de data e horario, sempre no formato 'YYYY-MM-DD hh:mm:ss'

  • TIMESTAMP

    Campo de data, sempre no formato de DATETIME, porém este tipo aceita apenas dados entre 1970 e 2038

Avançando em SELECT

Cláusulas

  • WHERE

    É o melhor e mais importante filtro para trazer dados mais específicos dentro de uma query

  • HAVING

    É semelhante à clausula WHERE, porém se refere a agregate functions

  • DISTINCT

    Esta cláusula seleciona apenas os valores distintos entre si (alternativa para GROUP BY)

  • EXISTS/ANY

  • AND/OR/NOT

    Ajuda o WHERE a filtrar ainda mais especificamente a consulta

  • ORDER BY column_name DESC/ASC

    Coloca os dados trazidos em ordem de acordo com a coluna selecionada

  • LIMIT amount

    Limita o número de dados trazidos pela query

  • SQL_SAFE_UPDATES = 0

    Remove controle de segurança para updates sem cláusula WHERE (evitar ativar)

  • ALIAS

    SELECT COUNT(*) AS alias_name FROM table_name

Operadores

  • <,>, <=,>=, =

    Operadores básicos da matematica

  • LIKE

    SELECT * FROM table_name WHERE column_name LIKE '%text_split%'

  • IN

    SELECT * FROM table_name WHERE column_name IN (column_item, column_item)

  • BETWEEN

    SELECT * FROM table_name WHERE column_name BETWEEN column_item_min AND column_item_max (aceita ORDER BY)

Agrgegate functions

  • MIN(column_name)

    SELECT MIN(column_name) FROM table_name

  • MAX(column_name)

    SELECT MAX(column_name) FROM table_name

  • COUNT(column_name)

    SELECT COUNT(column_name) FROM table_name WHERE data_comparison

  • AVG(column_name)

    SELECT AVG(column_name) FROM table_name

  • SUM(column_name)

    SELECT SUM(column_name) FROM table_name OBS: Não funciona com ORDER BY column_name

Funções em SQL

Funcões de String

  • CHAR_LENGTH(string)

    Retorna a quantidade de caracteres de uma determinada string

  • CONCAT(string, " ", string)

    A função CONCAT_WS("separador", string1, string2) concatena dados com um separador comum

  • FORMAT(number, slots)

    Formata um número com determinado número de casas após a vírgula e arredonda de acordo com a regra da matematica

  • INSTR(string, substring)

    Retorna a posição da substring na string alvo

  • LCASE(string) / UCASE(string)

    Transforma o conteúdo da string em caixa baixa ou caixa alta, respectivamente

  • LEFT(string, number) / RIGHT(string, number)

    Extrai determinado número de caracteres de uma string, a começar pela esquerda (LEFT) ou pela direita (RIGHT)

  • REPLACE(column_name, string_before, string_after)

    Substitui determinado texto de uma coluna

  • SUBSTR(column_name, index, amount)

    Extrai determinada quantidade de caracteres (amount) de uma string, começando pelo index

Funcões de Number

  • COUNT(column_name)

    Conta o número de ocorrências em uma determinada coluna

  • CEIL(column_name) / FLOOR(column_name)

    Arredonda para cima/baixo

  • MAX(column_name) / MIN(column_name)

    Retorna o maior/menor valor de uma determinada coluna

  • SUM(column_name)

    Retorna a soma dos valores de uma determinada coluna

Funcões de Date

  • ADDDATE(column_name, date_amount) / ADDTIME(column_name, time_amount[yyyy-mm-dd hh:mm:ss])

    Adiciona ou remove dias, meses, anos, horas, minutos e segundos em um campo do tipo DATE/TIMESTAMP

  • DATEDIFF(date1, date2)

    Retorna a diferença entre duas datas, em dias

  • DATE_FORMAT(date, format[%d, %m, %Y])

    Formata uma data para o modelo desejado

  • DAY(), MONTH(), YEAR()

    Retorna um número equivalente ao dia, mês, ano em determinada string no formato YYYY-mm-dd hh:mm:ss

  • DAYOFWEEK(), DAYOFYEAR(), WEEKOFYEAR()

    Retorna um número equivalente ao dia da semana, dia do ano e semana do ano em determinada string no formato YYYY-mm-dd hh:mm:ss
