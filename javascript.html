Features da linguagem

Variáveis

- São declaradas com as palavras reservadas "var" (hoje em dia é a menos utilizada), "let" e "const".

- Tipos de variáveis: number, string, boolean, array, object, null, undefined, Symbol

- Variáveis não podem iniciar com números, não podem ser palavras reservadas (var, let, const, for, class, etc..), não podem conter acentos

- O comando "typeof variavel" retorna o tipo da variavel em questao

O tipo boolean

Este tipo retorna verdadeiro ou falso. É usado para definir rumos em estruturas condicionais.

Os seguintes casos retornam falso: false, 0, NaN, null, undefined, '' ou "" ou ``

Diferença entre var, let e const

var é o tipo de declaração usado no começo do javascript. Ainda funcional, possui escopo global, isto é, pode ser acessado de onde quer que esteja. Podem ter seus valores reestabelecidos e podem ser redeclarados.

let é usada para variáveis que têm escopo de bloco, o que quer dizer que só tem seu valor dentro de um bloco de código, limitado por {}, além disso; podem ter seus valores reestabelecidos mas não pode ser redeclarado.

const é usado para variáveis que deverão permanecer constante. Possui escopo de bloco, assim como o tipo let mas não podem ter seus valores reestabelecidos e não podem ser redeclarados.

Estruturas condicionais

- Sâo blocos de códigos para executar determinadas ações de acordo com a veracidade de uma determinada condição

- Operadores de comparação: &&, ||, <, >, <=, >=, ==, === (usa-se o sinal de ! para negar uma condição)

- Podem ser do tipo if-else ou do tipo switch-case

Estrutura if-else

if(condição) {
bloco de comando 1;
} else if (condição 2) {
bloco de comando 2;
} else {
bloco de comando 3;
}

Estrutura switch-case

switch (condição) {
case 1:
bloco de comando 1; break;
case 2:
bloco de comando 2; break;
default:
comando caso !1 e !2;

Funções

- Parâmetros são os valores requeridos ao chamar uma função

- Argumento são os valores passados para uma função

- Funções podem serem usadas como argumentos para outras funções

- Uma função sempre retorna alguma coisa. Este retorno pode ser qualquer tipo de dados (objetos, variaveis, funções, arrays). Se não for definido, retorna 'undefined'

- Variáveis criadas dentro de funções têm escopo local e não podem ser acessadas de fora dela

- Callback são funções executadas por consequência de uma outra função, estas funções podem ser síncronas ou assíncronas

Objetos

- Objetos são conjuntos de variáveis (propriedades), independente de seus tipos

- Se uma dessas variáveis (propriedades) for uma função ela será chamada de método

- Você pode acessar uma propriedade de um objeto dentro dele mesmo utilizando a palavra-chave this.

- Métodos (funções) podem ser abreviados dentro de objetos.

- A propriedade "metodo: function () {}" pode ser abreviada para "metodo () {}"

- Existem objetos nativos do Javascript, como por exemplo o Math. Algumas referências podem citar objetos nativos como bibliotecas

- (no VSCode) Para acessar as propriedades do objeto Math, em um arquivo javascript, digite "Math." e depois "ctrl+espaço" e o VSCode irá exibir todas as propriedades, inclusive funções.

- É possível criar uma propriedade depois do objeto já ter sido criado utilizando o comando objeto.novaPropriedade = 'valor';

- Uma propriedade existente pode ser alterada a qualquer momento utilizando objeto.propriedadeExistente = 'novo valor';

- IMPORTANTE: objetos herdam propriedades do objeto pai. Geralmente este objeto pai é o Object, nativo do javascript.

- Este objeto pai dá origem a qualquer tipo de variavel, como string, number e object e cada objeto pai tem suas propriedades e métodos.

Atribuições

- É possível atribuir valor a uma variável de outras formas além de x = 'valor'

- Uma dessas é o auto incremento, que funciona da seguinte forma: x += 2, este formato representa que o valor x recebe ele mesmo + 2

- Também é possível atribuir valor utilizando o operador ternário: x = (condição) ? (true) : (false)
