Comandos Básicos

Comandos gerais

  • cd Entra em uma determinada pasta
  • cd .. Volta uma pasta
  • pwd Informa o nome da pasta
  • touch Cria um arquivo
  • mkdir Cria uma pasta
  • clear Limpa o prompt
  • ls Lista os itens que estão na pasta atual
  • less Exibe o texto do arquivo
  • mv 'a' 'b' Move o arquivo 'a' para o destino 'b'
  • code . Abre o VSCode
  • rmdir 'nome' Apaga uma pasta

Comandos Git

  • init Iniciar o versionamento em uma pasta
  • status Checar a situação dos arquivos
  • diff Para verificar o que mudou nos arquivos
  • add . Para salvar os arquivos editados
  • checkout -b 'branch' Para trocar para uma determinada branch
  • checkout 'file_name' Remove alterações
  • commit -m "comentário" Para enviar as alterações
  • branch nome Cria uma nova branch
  • branch Verifica lista de branches
  • merge nome Faz o merge da branch selecionada
  • push Empurra atualizações para a branch principal
  • pull Puxa as últimas atualizações da branch principal
  • clean -f Exclui arquivos não versionados

Comandos VSCode

  • init Iniciar o versionamento em uma pasta
  • status Checar a situação dos arquivos
